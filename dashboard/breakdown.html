Where everything belongs

PRN / four tiers

The rule every agent follows without guessing. This is what keeps storage from turning back into a mess as projects multiply.

CodeGitHub
The source of truth for anything that is code. A repo on only one machine does not exist.
HotCX33 /root/projects, /srv
Working files, running services, active data. Under a few GB. Mirrored to the Storage Box every hour.
ColdStorage Box
Large data, old projects, permanent archives. Streamed on demand. This tier is the backup.
Ephemeralnowhere
node_modules, caches, build output. Regenerable, so it is excluded from backup by design.

The project system

PRJ / built today

Every project carries its own context so agents never start cold, and a page that regenerates hourly from facts that cannot lie.

Three files per project

  • AGENTS.mdWhat it is, stack, conventions, boundaries, storage. Durable.
  • CLAUDE.mdTwo-line router to AGENTS.md. Works for any AI tool.
  • STATUS.mdNow, next, decisions, blocked. The handoff note.

Automatic, hourly

  • ScanLast commit, uncommitted work, idle days, disk vs budget, service state.
  • Overview pageEvery project, staleness and overruns flagged. On Mac and phone.
  • Scaffoldnew-project.sh creates a ready project in seconds.

Safety nets and secrets

SEC / how you sleep

If something breaks

  • hourlyServer mirrors to the Storage Box, deletes kept 30 days.
  • verifiedThe old server is 44.1 GiB archived, every file checksum-matched.
  • recommendedStorage Box daily snapshots, a 2-minute toggle in your console.

Secrets

  • DopplerOne dashboard, per-project scoped tokens. An agent in one project cannot read another's keys.
  • Never on diskSecrets inject at runtime, never committed, never backed up in plaintext.
  • 1PasswordStays home for your human passwords and logins.
